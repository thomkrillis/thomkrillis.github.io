Who made this?

My name is Bobby Yankou. The name Thom Krillis was born a few years ago when I audited a course on interactive design and had to create my first Github and Twitter accounts. I did not want to use my actual name, since at the time I did not know what I would end up doing with these accounts. Eventually, I started to use the Github account professionally, and the fake name has stuck.

For the majority of my life, I have been in the band, No One and the Somebodies. For the last couple of years of my life, I have been working part-time at The Hackerati as a software engineer.

This site is just for fun.
